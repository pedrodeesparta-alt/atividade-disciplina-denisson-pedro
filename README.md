## 📖 Descrição do Projeto

Este projeto consiste em um formulário HTML desenvolvido durante as aulas da disciplina.

O sistema possui:
- formulário de cadastro
- seleção de combustível
- checkbox
- seleção de fabricante
- campo de sugestões
- tela de sucesso após envio

Também foi aplicado um visual em modo escuro (dark mode) utilizando CSS.

---

## ▶ Como Executar

1. Abra o arquivo `formulario.html`
2. Preencha o formulário
3. Clique em "Enviar"
4. O sistema abrirá a tela `sucesso.html`
